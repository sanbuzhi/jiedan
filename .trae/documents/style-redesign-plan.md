# 客户端样式复刻详细计划

## 目标
将移动端客户端的样式完全复刻小程序端的视觉设计，包括颜色、布局、间距、字体等。

## 小程序样式特点总结

### 通用设计规范
- **主题色**: #07c160 (绿色)
- **背景色**: #f5f7fa (浅灰)
- **卡片圆角**: 24rpx → 12px
- **按钮圆角**: 12rpx → 6px
- **阴影**: 0 4rpx 20rpx rgba(0, 0, 0, 0.05)
- **字体**: 无衬线, 30rpx≈15px, 28rpx≈14px

### 页面样式差异分析

#### 1. 登录页 (login.wxss) - 严重差异
**小程序样式**:
- 整个容器绿色渐变背景 `linear-gradient(180deg, #07c160 0%, #06ad56 100%)`
- Logo图片 160rpx×160rpx (80px×80px)
- 白色登录卡片
- 功能列表带绿色圆形勾选图标
- 登录按钮绿色 #07c160

**当前实现问题**:
- 背景不是全屏绿色渐变
- Logo尺寸和样式不匹配
- 按钮样式不统一

#### 2. 个人中心 (profile.wxss) - 严重差异
**小程序样式**:
- 用户卡片绿色渐变背景
- 头像120rpx(60px)带白色边框
- 积分区域用分隔线
- 推荐码区域带复制/分享按钮
- 推荐树支持嵌套列表显示
- 弹窗样式统一（遮罩+圆角卡片）

#### 3. 积分页面 (points-list.wxss, points-exchange.wxss)
- 积分卡片样式
- 积分记录列表
- 兑换物品卡片

#### 4. 需求流程页面 (step.wxss, step_all.wxss, ui_gallery.wxss, approve.wxss, suggestion.wxss)
- 步骤指示器样式
- 表单项样式
- 按钮样式

## 样式转换规则
- rpx → px: 除以2 (rpx是相对于750px设计稿的单位)
- 颜色值直接复用
- 阴影使用相同的rgba值

## 验收标准
- 所有页面视觉与小程序保持一致
- 移动端适配良好
- 触摸目标尺寸 >= 44px

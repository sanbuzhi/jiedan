### 任务1: 后端推荐码生成、二维码与积分兑换接口 ✅
**任务描述**: 实现推荐码自动生成、二维码生成和积分兑换相关后端接口
- [x] 1.1 修改用户注册/登录逻辑
  - 用户首次登录时自动生成6位推荐码
  - 推荐码格式：大写字母+数字组合
  - 确保推荐码唯一性（数据库唯一索引）
- [x] 1.2 创建积分兑换物品数据表 `exchange_items`
  - 字段：id, name, description, image_url, points_required, cash_price, stock, is_active, created_at
- [x] 1.3 创建积分兑换订单表 `exchange_orders`
  - 字段：id, user_id, item_id, points_used, cash_paid, status, created_at
- [x] 1.4 实现积分兑换接口 `POST /api/v1/points/exchange`
  - 请求参数：item_id, use_cash（是否使用现金补充）
  - 校验积分余额
  - 计算现金差额（如需）
  - 生成兑换订单
- [x] 1.5 实现获取兑换物品列表接口 `GET /api/v1/exchange-items`
- [x] 1.6 实现获取用户兑换订单接口 `GET /api/v1/points/exchange-orders`
- [x] 1.7 实现推荐码二维码生成接口 `GET /api/v1/users/referral-qrcode`
  - 调用微信 `wxacode.getUnlimited` 接口
  - 参数：page=`pages/index/index`, scene=`referral_code={推荐码}`
  - 返回二维码图片数据（Base64或URL）

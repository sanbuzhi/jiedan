# 图片资源目录

此目录用于存放小程序所需的图片资源。

## 需要的图片文件

### TabBar图标
- `home.png` - 首页未选中图标 (81x81px)
- `home-active.png` - 首页选中图标 (81x81px)
- `profile.png` - 我的未选中图标 (81x81px)
- `profile-active.png` - 我的选中图标 (81x81px)

### 其他图片
- `logo.png` - 应用Logo (建议320x320px)
- `default-avatar.png` - 默认头像 (建议200x200px)
- `share-cover.png` - 分享封面图 (建议500x400px)

## 图标设计规范

1. TabBar图标建议使用81x81px的PNG格式
2. 图标颜色：
   - 未选中：#999999 (灰色)
   - 选中：#07c160 (绿色)
3. 图标风格保持一致，使用线性或面性图标

## 获取图标

可以使用以下方式获取图标：
1. [阿里巴巴矢量图标库](https://www.iconfont.cn/)
2. [IconPark](https://iconpark.oceanengine.com/)
3. 自行设计

# 酒店管理系统需求文档
版本号：V1.0  
编写日期：202X年XX月XX日  
适用对象：10-500间房的商务酒店、精品民宿、快捷酒店

---

## 1. 需求概述
### 1.1 项目背景
当前主流中小到中型酒店普遍存在**运营效率低、客户体验差、成本管控弱、数据支撑不足**的痛点：
- 前台依赖Excel/纸质管理房态、预订、押金，易出现错房、漏房、账目核对耗时30分钟以上的问题；
- 入住/退房流程需手工填写表单、验证证件，平均耗时超10分钟；
- 库存（客房布草、易耗品）、采购无数字化台账，易积压或断供；
- 财务报表需人工汇总，滞后性强，无法辅助实时运营决策。

### 1.2 项目目标
通过全流程数字化改造，实现：
1. **效率提升**：入住/退房压缩至3分钟内，房态/财务实时同步，日终/月终报表自动生成；
2. **体验优化**：支持线上线下多渠道预订、自助入住/退房（可选扩展）、会员权益自动兑现；
3. **成本管控**：易耗品/布草库存预警、采购全流程审批、能耗（可选扩展）与入住关联分析；
4. **数据决策**：提供入住率、客单价、复购率、渠道转化率等多维度可视化报表。

### 1.3 适用范围
#### 基础版
覆盖10-200间房的标准业务流程：预订、入住、退房、房态、宾客、财务、库存、报表管理。
#### 扩展版
覆盖200-500间房+高体验需求：多渠道直连（OTA、微信小程序/公众号）、自助终端对接、会员营销、能耗管理、智能门锁对接、公安身份核验系统对接。

## 2. 功能模块清单
### 2.1 前台接待管理（核心模块）
| 子模块名称       | 详细功能描述                                                                 |
|------------------|------------------------------------------------------------------------------|
| 散客预订         | 手动选择入住/退房日期、房型、房间数、房价（支持门市价/协议价/会员价切换）、宾客信息、押金金额；生成预订确认单（支持打印/短信/微信推送）；预订取消/修改（可设置取消规则，如提前24小时免费、12-24小时扣50%、12小时内全扣）；预订转锁房。 |
| 团队/协议客户预订 | 批量选择房间、选择团队/协议客户名称（自动匹配协议房价）、设置入住/退房统一时间、团队负责人信息、团队总押金/分摊押金、备注事项；生成团队入住通知单；批量取消/修改（仅允许修改未入住房间）。 |
| 入住登记         | 散客/团队/OTA预订（基础版需手动导入，扩展版自动同步）快速调取预订信息；证件识别（基础版手动录入，扩展版对接身份证阅读器、公安身份核验系统、健康码行程码接口）；分配房间（支持手动分配/系统智能分配：根据宾客偏好、房态优先级、空房距离优化）；收取押金（支持现金、POS、微信/支付宝扫码支付，生成押金单）；会员自动绑定/积分自动累计；打印入住单/房卡（扩展版对接智能制卡机）。 |
| 退房结算         | 输入房间号/刷房卡/扫宾客码快速调取宾客账户；自动统计住店期间所有消费（房费、小商品、餐饮、洗衣、加床、延时费等，基础版需手动录入餐饮/洗衣等，扩展版对接POS/餐饮/洗衣系统）；押金冲抵、多退少补（支持多种支付方式退款）；自动生成消费明细单、发票抬头预设与打印（电子发票对接扩展版）；房态自动切换为“空脏”；积分自动兑现（可选抵扣房费）；打印退房单。 |
| 房态管理         | 房态实时展示（颜色区分：空净=绿、空脏=黄、维修=红、住净=蓝、住脏=灰、预抵=紫、预离=橙、内部锁房/OTA保留房=黑）；单房/批量房态切换（如保洁打扫完“空脏”点“完成打扫”自动变“空净”）；房态筛选（按楼层、房型、房态类型、预抵/预离日期）；房态日志查询（谁、什么时候、因为什么修改了房态）；房间详情查看（面积、床型、朝向、设施、历史入住记录、维修记录）。 |
| 宾客账户管理     | 散客/团队/协议客户独立账户；消费明细实时录入与查询；押金充值/查询/退款；挂账权限设置（仅协议客户可设置，设置挂账额度、账期）；挂账还款管理；宾客标签管理（如“常客”“VIP”“过敏患者”“无烟房需求”）；宾客历史记录查询（入住次数、累计消费、常用房型、偏好设施）。 |
| 证件管理         | 基础版：证件信息手动录入、保存、导出（Excel/PDF）；扩展版：身份证阅读器自动扫描、识别、上传；对接公安旅馆业治安管理信息系统自动上传；证件有效期预警（如协议客户/常客身份证即将过期，提前30天提醒前台/负责人）。 |

### 2.2 线上预订/会员管理（可选扩展为基础版，推荐标配）

|------------------|------------------------------------------------------------------------------|
| 多渠道预订入口   | 微信小程序/公众号预订、酒店官网预订、第三方OTA直连（携程、美团、飞猪、同程等，扩展版需对接API网关）；预订入口显示实时房态、房价、剩余房间数、预订须知、取消规则、酒店介绍、房型图片/VR全景（VR扩展版）。 |
| 会员体系管理     | 会员注册（线上线下均可，支持手机号/微信/身份证注册）；会员等级设置（如银卡、金卡、白金卡、钻石卡，可自定义升级规则：消费金额/入住次数）；会员权益设置（如折扣房费、免费早餐、延时退房、优先入住、积分倍数、专属楼层）；会员积分管理（积分获取规则：消费1元积1分/入住1晚积50分，积分有效期设置：1年/2年/永久；积分兑换规则：兑换房费、小商品、免费早餐、优惠券）；会员优惠券管理（优惠券类型：满减券、折扣券、房型专属券；发放方式：注册赠送、生日赠送、节日活动、积分兑换；使用规则：有效期、适用房型、满减门槛、不可叠加）；会员标签与画像（标签同前台，画像包括年龄、性别、职业、消费习惯、入住偏好）。 |
| 自助入住/退房    | 对接自助终端机（一体机/壁挂机）；线上预订未到店宾客刷身份证/输入预订号+手机号+验证码完成身份核验、选择房间、支付押金、制卡/生成电子房卡（对接智能门锁）；线上预订已到店宾客刷身份证/输入房间号完成身份核验、自助续住/修改退房时间、自助查看消费明细、押金冲抵、多退少补（退款原路返回）、自助打印发票（电子/纸质）；自助终端实时同步后台数据、故障预警（提示管理员）。 |

### 2.3 客房管理（核心模块）
| 子模块名称       | 详细功能描述                                                                 |
|------------------|------------------------------------------------------------------------------|
| 保洁任务管理     | 系统自动生成“空脏房→空净房”“住脏房→住净房”“预离房退房后→空脏房”的保洁任务；按楼层、保洁员分配任务（支持手动调整）；保洁员移动端（APP/小程序）接收任务、查看房间详情、开始/完成/延迟任务（延迟需填写原因）；任务完成后前台/客房主管收到通知、可抽查验收；保洁任务日志查询（谁、什么时候、做了什么任务、完成时长）。 |
| 客房维修管理     | 保洁员/宾客/前台发现客房设施故障（如空调坏、马桶堵、灯不亮），通过移动端/前台系统提交维修工单；工单内容包括房间号、故障类型、故障描述、图片/视频、紧急程度（普通/紧急/非常紧急）；系统自动分配维修任务给维修工（按楼层/技能分配）；维修工移动端接收任务、查看工单详情、开始/完成/延迟任务（延迟需填写原因）、上传维修完成图片/视频、填写维修费用；任务完成后前台/客房主管/宾客收到通知、可验收确认；维修工单日志查询、维修费用统计、维修设施统计（哪些设施易坏、坏的频率）。 |
| 客房物品管理     | 每个房间建立物品清单（布草、易耗品、家具、电器、装饰品）；布草/易耗品消耗自动统计（如退房时系统自动扣除1套布草、2瓶矿泉水、1包纸巾；保洁员可通过移动端调整数量，如宾客多要1瓶矿泉水）；物品盘点（支持移动端扫码盘点，自动生成盘点差异表）；物品报损管理（布草/易耗品/家具电器损坏，填写报损单、上传图片/视频、客房主管审批后扣除库存）。 |

### 2.4 库存管理（核心模块）
| 子模块名称       | 详细功能描述                                                                 |
|------------------|------------------------------------------------------------------------------|
| 基础信息管理     | 供应商信息管理（名称、联系人、电话、地址、供应商品类、结算方式、账期）；商品信息管理（名称、分类、规格、单位、进价、售价、库存预警线（最低/最高）、存放位置）。 |
| 入库管理         | 采购订单到货后，填写入库单（关联采购订单）、核对商品数量/规格/质量、仓库管理员签字确认、系统自动增加库存；支持扫码入库（扩展版）；入库单打印、导出；入库日志查询。 |
| 出库管理         | 前台/客房/餐饮等部门申领商品，填写出库单（关联部门/申领人）、仓库管理员核对数量/规格、签字确认、系统自动减少库存；支持扫码出库（扩展版）；出库单打印、导出；出库日志查询。 |
| 库存盘点         | 支持日盘点/周盘点/月盘点/临时盘点；支持移动端扫码盘点（自动比对系统库存与实际库存）、生成盘点差异表；差异调整需填写原因、仓库主管/财务主管审批后生效；盘点单打印、导出；盘点日志查询。 |
| 库存预警         | 商品库存低于最低预警线时，自动生成采购建议单（建议采购数量=最高预警线-当前库存+日均消耗×采购周期）、提醒仓库管理员/采购负责人；商品库存高于最高预警线时，自动提醒仓库管理员/财务主管；预警方式：系统弹窗、短信、微信（扩展版）。 |

|------------------|------------------------------------------------------------------------------|
| 采购申请管理     | 各部门（前台、客房、餐饮、行政等）根据需求/库存建议单填写采购申请单（商品名称、分类、规格、单位、数量、预算金额、备注事项）；部门主管审批后提交给采购负责人；采购申请单打印、导出；采购申请日志查询。 |

| 采购付款管理     | 关联已入库的采购订单；填写付款申请单（供应商信息、订单号、应付金额、已付金额、本次付款金额、付款日期、付款方式）；财务主管审批后提交给出纳；出纳付款后标记付款状态为“已付款”；付款申请单/付款凭证打印、导出；采购付款日志查询；应付账款统计（按供应商、按账期）。 |

| 利润管理         | 自动计算每日/每周/每月/每季度/每年的利润（利润=总收入-总支出）；生成利润日报表/周报表/月报表/季度报表/年度报表；报表打印、导出（Excel/PDF）；利润趋势分析（图表展示：折线图、柱状图）。 |
| 对账管理         | 自动与POS机、微信/支付宝、银行账户、OTA平台对账（扩展版对接API自动对账，基础版需手动导入数据对账）；自动生成对账差异表；差异调整需填写原因、财务主管审批后生效；对账单打印、导出（Excel/PDF）；对账日志查询。 |
| 发票管理         | 基础版：手动录入发票抬头、税号、金额、开具日期；支持打印纸质发票；生成发票开具记录；扩展版：对接电子发票平台（如百望、航天信息）；支持线上线下申请开具电子发票（自动推送到宾客邮箱/微信）；支持发票红冲；生成发票开具/红冲记录、统计已开具/未开具发票金额。 |

### 2.7 报表分析（可选扩展为基础版，推荐标配）
| 子模块名称       | 详细功能描述                                                                 |
|------------------|------------------------------------------------------------------------------|
| 运营报表         | 入住率报表（按时间、按房型、按楼层、按渠道）；客单价报表（按时间、按房型、按楼层、按渠道）；复购率报表（按时间、按会员等级）；渠道转化率报表（各OTA/线上入口的访问量、预订量、入住量、转化率）；预抵率/预离率报表；平均出租天数报表；房间收入贡献报表（每个房间的累计收入、累计出租天数、收入贡献占比）；所有报表支持图表展示（折线图、柱状图、饼图、环形图）、数据筛选、自定义时间范围、导出（Excel/PDF）。 |
| 财务报表         | 除财务管理模块已有的收入、支出、利润报表外，扩展版支持资产负债表、现金流量表的自动生成（对接标准会计科目）。 |

| 会员报表         | 会员注册量报表（按时间、按注册渠道）；会员等级分布报表；会员消费报表（按会员等级、按时间、按消费类型）；会员复购报表；会员优惠券使用报表（发放量、使用量、使用率、带动消费金额）。 |
| 自定义报表       | 支持用户根据自身需求自定义报表（选择数据源、字段、图表类型、数据筛选条件、时间范围）；支持保存自定义报表模板、分享给其他用户。 |

### 2.8 系统管理（核心模块）

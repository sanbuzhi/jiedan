## 1. 需求概述
### 1.1 项目背景
当前国内中小规模（10-500间房）经济型、商务型、精品型酒店/连锁门店普遍存在以下痛点：
1. 房态、订单、会员管理依赖纸质或Excel，信息同步滞后、数据易丢失
2. 前台收银效率低，手工核对押金、房费、账务易出错
3. 房务、保洁协作断层，查房、打扫反馈不及时影响入住/退房
4. 缺乏数据分析能力，无法精准制定营销策略
5. 客户服务触达渠道单一，会员体系不完善，复购率低

为解决上述问题，需开发一套**覆盖门店前台、后台管理、客户自助全场景**的云端/本地化部署可选的酒店管理系统。

### 1.2 建设目标
1. **全流程信息化**：实现预订、入住、房务、收银、退房、会员、营销、库存、报表全流程数字化闭环
2. **降本提效**：前台收银效率提升40%以上，房务协作响应时间缩短60%，人工账务核对成本降低80%
3. **数据驱动决策**：提供实时、多维度的经营、客群、库存分析报表
4. **优化客户体验**：支持多渠道预订、自助入住/退房、线上选房、会员积分兑换等功能
5. **降低系统成本**：可选云端SaaS模式（按需付费），适配中小酒店预算；本地化部署需支持主流操作系统

### 1.3 适用范围
- **门店覆盖**：10-500间房的单店或直营/加盟型连锁门店
- **业务场景**：
  - 门店：前台接待、收银、房态管理、房务排班、库存管理
  - 总部（连锁模式）：多门店权限管理、统一营销、统一财务结算、集中数据分析
  - 客户：多渠道预订、会员服务、自助操作
- **技术环境**：
  - 云端模式：支持Chrome、Firefox、Safari、Edge主流浏览器，适配移动端微信/支付宝小程序、H5
  - 本地化模式：支持Windows Server 2016及以上、CentOS 7及以上操作系统，MySQL 8.0及以上数据库

## 2. 功能模块清单
### 2.1 门店前台管理模块
#### 2.1.1 订单管理子模块
- 多渠道订单接入（直连携程/美团/飞猪/同程艺龙、自有小程序/H5、电话/到店预订）
- 订单查询（按房号、姓名、手机号、身份证号、订单状态、渠道、日期范围、预订人/入住人筛选）
- 订单操作（修改入住/退房日期、调整房型、取消订单、拒绝/接受取消、补录订单、转单、拆分/合并订单、添加备注）
- 订单预警（超期未入住、超期未退房、入住人信息不全、会员等级/权益异常）
- 房态匹配（预订时自动锁定可用房、可选房态优先级配置、房号手动分配/自动分配规则设置）

#### 2.1.2 入住管理子模块
- 身份核验（身份证OCR识别、人脸比对对接公安系统[可选，需单独申请接口]、入住人信息自动填充、未成年人入住监护人信息登记、外宾/港澳台胞入住信息登记）
- 入住登记（选择/匹配房型房号、读取身份证OCR生成电子登记单、打印纸质登记单、押金缴纳方式配置、押金打印、电子签名[可选]、房卡制作/绑定[对接主流房卡制卡机]）
- 换房管理（换房原因选择、房态同步调整、房费/押金差额计算、换房单打印）
- 续住管理（续住日期选择、房态锁定、续住房费计算、押金补缴/退还、续住单打印）

#### 2.1.3 退房管理子模块
- 快速退房（一键退房、预授权完成/撤销、实时房态同步）
- 常规退房（身份核验、房卡回收、消费明细查询/核对、延迟退房审批[需配置权限]、延迟退房费用计算、消费/退款/押金退还明细汇总、电子/纸质账单打印、发票申请/开具[对接金税四期接口]、会员积分自动累积/扣减、退房评价提醒发送）
- 预授权管理（预授权冻结、解冻、完成、撤销记录查询）

#### 2.1.4 收银管理子模块
- 支付方式配置（现金、微信/支付宝/银联扫码、银行卡POS、储值卡、优惠券、积分抵现、企业签单、挂账）
- 收银操作（入住押金收取、退房消费结算、挂账/签单还款、日常收入/支出记录、发票红冲/作废、退款操作[含原路退回、现金退回、转入储值卡]、收银权限配置）
- 交接班管理（交接班时间配置、交接班前强制盘点现金、交接班报告自动生成、交接班签字确认[电子/纸质]、交接班历史查询）
- 账务查询（按日期、时间、收银员、支付方式、账单状态筛选、单笔账单明细查询、账单导出）

### 2.2 房务管理模块
#### 2.2.1 房态管理子模块
- 基础房态设置（空房、脏房、维修房、自用房、锁房、预订中、已入住、已退房未查房、已查房未打扫、已打扫待复查、已确认入住房态）
- 房态操作（一键切换房态、批量切换房态、房态备注添加、维修房/自用房/锁房时间设置、房态日志查询）
- 房态同步（前台、房务、保洁、后台管理房态实时同步、多渠道预订房态实时同步）

#### 2.2.2 房务排班子模块
- 员工基础信息管理（房务员/保洁员姓名、工号、手机号、身份证号、技能标签、排班权限配置）
- 排班操作（周/月排班视图、批量排班、复制上周/上月排班、临时调班、请假申请/审批、排班导出）
- 排班报表（员工工时统计、请假统计、工作量统计、排班导出）

#### 2.2.3 房务工单子模块
- 工单生成（前台退房/换房自动生成打扫工单、维修房自动生成维修工单、日常巡检工单手动生成、客户投诉工单自动/手动生成）
- 工单分配（自动分配规则设置[按区域、技能标签、工作量]、手动分配工单、工单转派）
- 工单执行（工单接收/拒绝、工单进度更新、工单完成拍照/上传、工单备注添加）
- 工单审核（打扫工单查房审核、维修工单验收审核、工单评价、工单超时预警）
- 工单报表（工单完成率统计、工单超时率统计、员工工作量统计、工单类型统计、工单导出）

#### 2.2.4 房务库存管理子模块
- 库存基础信息管理（物品分类、物品名称、规格、单位、库存上限/下限、存储位置、入库/出库权限配置）
- 入库操作（采购入库、调拨入库、退货入库、入库单生成、入库单打印/电子存档、入库记录查询）
- 出库操作（房务领用出库、调拨出库、报废出库、出库单生成、出库单打印/电子存档、出库记录查询）
- 库存盘点（周/月/季度盘点计划设置、盘点单生成、盘点结果录入、盘盈/盘亏处理、盘点报告导出）
- 库存预警（库存低于/高于上限预警、库存预警消息推送）

### 2.3 会员管理模块
#### 2.3.1 会员等级设置子模块
- 等级基础信息设置（等级名称、等级图标、升级/保级规则[按积分、按消费金额、按入住次数、按周期]、降级规则设置）
- 等级权益配置（房价折扣[按房型/时段/节假日]、积分倍率、延迟退房时长、免费早餐份数、免费房型升级次数、生日/节日礼包、专属客服、预订优先权、免费取消时长）

#### 2.3.2 会员信息管理子模块
- 会员注册（手机号注册、微信/支付宝授权注册、身份证OCR注册、协议客户批量注册、企业对公账户绑定）
- 会员信息查询（按姓名、手机号、身份证号、会员卡号、会员等级、渠道、日期范围筛选）
- 会员信息修改（基础信息修改、密码重置、手机号更换、等级调整、积分调整、标签添加/删除、会员冻结/解冻）
- 会员档案（消费记录查询、积分明细查询、入住记录查询、评价记录查询、投诉记录查询）

#### 2.3.3 会员积分管理子模块
- 积分规则设置（消费积分规则[按支付方式/房型/时段/节假日]、入住积分规则、评价积分规则、推荐积分规则、生日/节日签到积分规则、积分抵现规则、积分有效期设置）
- 积分操作（积分自动累积/扣减、积分手动调整、积分冻结/解冻、积分清零规则设置）
- 积分明细查询（按会员、日期范围、积分类型筛选、积分明细导出）

#### 2.3.4 会员储值卡管理子模块
- 储值卡类型设置（卡名称、卡面值、充值赠送规则、有效期设置、使用范围设置）

- 短信/消息推送（短信模板设置、微信/支付宝小程序/H5消息模板设置、推送内容编辑、推送对象选择、推送时间设置、推送记录查询/导出）

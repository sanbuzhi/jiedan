## 需求概述
### 1.1 项目背景
需求方是一家独立运营的线下美妆店，主营彩妆、护肤、美妆工具、小样/试用装、套盒等品类，目前面临**人工记账繁琐易出错、库存效期管理混乱、会员维护效率低、销售数据统计不清晰**等问题，需要一套专门适配美妆店业务的网站版后台管理系统，实现业务的数字化、标准化、高效化。
### 1.2 项目目标
1. **核心业务数字化**：覆盖商品、库存、采购、销售、会员、财务全流程，替代人工纸质/Excel记账
2. **美妆专属管理**：针对小样、套盒、特殊批号、效期高敏感性做适配
3. **销售与会员激活**：提供多维度销售统计、会员标签/积分/优惠券体系，辅助决策与客户留存
4. **基础扩展性**：预留小程序/微商城、扫码枪/收银机对接接口
### 1.3 核心约束
1. 网站版支持Chrome、Safari、Edge最新3个版本浏览器
2. 系统需支持多账号同时登录操作（但需权限隔离）
3. 单店版本支持商品SKU≤10000、会员≤50000
4. 开发周期≤60天，上线后免费维护期≥3个月

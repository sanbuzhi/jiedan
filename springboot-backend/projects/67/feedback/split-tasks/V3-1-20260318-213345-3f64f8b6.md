## 1. 项目技术规格
### 技术栈
- **后端**：Java 17 + Spring Boot 3.2.5 + MyBatis Plus 3.5.5 + Swagger 3.0.0 + JWT 0.11.5 + Hutool 5.8.26
- **前端**：Vue 3.4.27 + Element Plus 2.7.3 + Vite 5.2.11 + ECharts 5.5.0 + xlsx 0.18.5 + jspdf-autotable 3.5.31 + Axios 1.7.2
- **数据库**：MySQL 8.0.36 + Redis 7.2.4（可选热点缓存）
- **服务器**：Nginx 1.24.0 + Spring Boot内置Tomcat 10.1.20

### 代码生成规则
- **依赖版本**：严格遵循上述技术栈版本，禁止随意升级
- **JWT配置**：
  - 签名算法：HS256
  - 有效期：7天
  - 刷新有效期：30天
  - 密钥：随机生成32位字符串，生产环境需配置在环境变量中
- **MyBatis Plus配置**：
  - 自动填充：create_time、update_time
  - 逻辑删除：deleted字段（0未删除，1已删除）
  - 分页插件：最大单页500条记录
- **数据库连接配置**：
  - 字符集：utf8mb4
  - 连接池：HikariCP
  - 最小空闲连接：5
  - 最大连接数：20
- **文件上传配置**：
  - 允许格式：JPG、PNG
  - 单张大小：≤2MB（商品图片）、≤1MB（等级图标、店铺logo）
  - 最多上传：商品5张、其他1张
  - 存储路径：本地存储可配置绝对路径，支持后续扩展OSS
